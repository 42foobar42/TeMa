How to play: This game is similar to Tetris. Use the arrow keys to move the stones side or downward. Generate equations to get points and remove the stones. Equations can be generated vertically, horizontally and diagonally.

This app is offline-capable. Create a shortcut on your screen and play even when you're not online.

Created by Alexander Boll. This game is slightly ajar to Mathris. In addition, in the application the Framework Math.js is used. The play and pause icon was created by Daniele De Santis
